## 为什么重要

PON 设备多厂商混插（OLT、ONU 常来自不同供应商）。互通测试是：
- 协议条款 → 可重放报文时序 + Pass/Fail 判据；
- 固件升级 / 芯片选型的回归基线；
- 异厂商组合时定位「谁不合规」的共同语言。
