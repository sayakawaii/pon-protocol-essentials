# PON 光功率预算与光链路（ODN class）

> ODN 是无源的：光从 OLT 出发，经馈线、分光器、配线到 ONU，一路衰减。**光功率预算**决定了能挂多少 ONU、拉多远。ITU-T 用 **光路损耗（OPL）等级**（B+/C+/N1/N2/E1/E2）规定允许的衰减范围，设备据此设计发光与收光灵敏度。依据 G.9807.1 Annex B（§B.6.1、表 B.6.1 / B.9.2 / B.9.3 / B.9.4）。

## 1. 链路预算基本式

```
可用预算 (dB) = 发射光功率(min) − 接收灵敏度
ODN 损耗     = 馈线/配线光纤损耗 + 分光器损耗 + 接头/熔接损耗 + 余量
要求: 接收灵敏度 ≤ 发光 − ODN损耗 ≤ 过载点(overload)
```

- **太弱**（损耗大）→ 收光低于灵敏度 → 误码/掉线；
- **太强**（损耗小）→ 收光高于**过载点** → 接收机饱和甚至损坏（近端 ONU 风险）；
- 标准给的不是单一值，而是**衰减范围 [min, max]**——同一 PON 既要兼容最远又要兼容最近的 ONU。

## 2. 光路损耗（OPL）等级（表 B.6.1 / B.9.2）

| OPL 等级 | 衰减范围 (dB) | 波长集 | 说明 |
|----------|--------------|--------|------|
| **B+** | 13 – 28 | 基本/兼容 GPON | 沿用 GPON B+ |
| **C+** | 17 – 32 | | 沿用 GPON C+ |
| **N1**（Nominal1） | 14 – 29 | 基本波长集 | XGS-PON 标称 |
| **N2**（Nominal2） | 16 – 31 | 基本波长集 | |
| **E1**（Extended1） | 18 – 33 | 基本波长集 | 扩展 |
| **E2**（Extended2） | 20 – 35 | 基本波长集 | 最大覆盖 |

> ⚠️ 若实际架构损耗**低于**该等级最小值，**必须在 ODN 加光衰减器**，保证最小信道插损，**防止收光过载损坏接收机**（表 B.6.1 注）。

## 3. 距离与差分

| 参数 | 值 | 含义 |
|------|-----|------|
| **DD20** | 20 km | S/R 与 R/S 间最大光纤距离（20 km 级） |
| **DD40** | 40 km | 40 km 级 |
| **最大差分光路损耗** | 15 dB | 同一 PON 上最近与最远 ONU 的损耗差上限 |
| 最大光通道代价 | 1.0 dB (DD20) / 2.0 dB (DD40) | 色散等代价 |
| 设备反射率 @R/S | < −20 dB | 限制反射 |
| 光纤类型 | ITU-T G.652 | 标准单模 |

> **差分**很关键：最近 ONU 收很强、最远 ONU 收很弱，OLT 突发接收机需在 15 dB 动态范围内逐突发适配（见 [测距与激活](gpon-g984/ranging-activation.md) 的 burst 概念）。

## 4. 波长（基本波长集，XGS-PON）

| 方向 | 波长 | 接口 |
|------|------|------|
| 上行（ONU→OLT） | **1260 – 1280 nm** | Oru / Old 收 |
| 下行（OLT→ONU） | ~**1577 nm**（1575–1580 区） | Old 发 / Ord 收 |

> 上下行波长分离 → 与 GPON（1490/1310 nm）共存于同一 ODN（见 [共存](hsp-g9804/overview.md)）；波长不对称也是 [时间同步](../05-operations/time-synchronization.md) 误差源之一。

## 5. 预算估算示例（概念）

```
假设 N1 等级, 预算 ≈ 29 dB
  20 km 光纤 @0.3 dB/km ............ 6.0 dB
  1:64 分光器 .................... ~20.0 dB
  接头/熔接/余量 .................. ~3.0 dB
  合计 ............................ ~29 dB  → 恰在 N1 上限
```

- 想 1:128 或更远 → 需更高等级（E1/E2）、开 [FEC](fec-principles.md) 增益、或加 Reach Extender。
- FEC 净增益（RS 典型 5~7 dB）可直接折算成额外分光比或距离。

## 6. 工程检查清单

1. ONU 实测收光是否落在 **[灵敏度, 过载]** 区间；
2. 近端 ONU 是否触发**过载**（必要时加衰减器）；
3. 最近/最远 ONU 损耗差是否 ≤ **15 dB**；
4. 选用的 OPL 等级与设备光模块标称是否匹配；
5. 收光低/高告警门限按等级设置（见 [告警速查 ANI-G](../02-omci/alarm-reference.md)）。
