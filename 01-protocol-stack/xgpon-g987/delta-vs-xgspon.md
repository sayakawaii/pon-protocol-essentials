# XG-PON 与 XGS-PON 的差异

> XG-PON（NG-PON1，ITU-T G.987.x）与 XGS-PON（G.9807.1）TC 层高度同源。本篇以「差异」方式呈现，避免重复。

## 核心差异

| 维度 | XG-PON (G.987.x) | XGS-PON (G.9807.1) |
|------|------------------|--------------------|
| 上行速率 | 2.488 Gbit/s | 9.953 Gbit/s（对称 10G） |
| 下行速率 | 9.953 Gbit/s | 9.953 Gbit/s |
| 定位 | NG-PON1，10G 非对称 | 对称 10G，主流升级路径 |
| TC 层 | G.987.3 | G.9807.1 Annex C（继承 G.987.3 框架） |
| 上行波长 | 1270 nm | 1270 nm |

> G.9807.1 大量引用 G.987.3 的 TC 层定义（帧、PLOAM、DBA、状态机基本一致），主要差异在上行 PMD 速率。互通测试（BBF TR-309 / TP-255）经常将 XG-PON / XGS-PON / 25GS-PON 并列引用。
